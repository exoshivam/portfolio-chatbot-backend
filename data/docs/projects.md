# Projects

## AI Career Mentor – LLM-Based Interview & Career Guidance System

Developed an AI-powered mock interview and career guidance platform using LLMs with multi-step reasoning workflows (LangChain).

### Key Features

- Built an adaptive interview engine that dynamically generates domain-specific questions and follow-ups based on user responses.
- Implemented LLM-based answer evaluation and automated scoring system with final hire/no-hire decision logic.
- Integrated SerpAPI for real-time project recommendation with GitHub links of the projects.
- Integrated course recommendations based on user's weak area with SerpAPI.

### Tech Stack

- LangChain, LangGraph, LLMs, SerpAPI, Python

## Rural School Attendance System (Full-Stack | IoT | AI)

Built an IoT-enabled smart attendance system using ESP32 and RFID, integrated with AI-based facial recognition for verification.

### Key Features

- Developed secure REST APIs and MongoDB-based storage for real-time attendance logging and tracking.
- Designed role-based dashboards (Admin, Teacher) with analytics for attendance trends and absentee detection.
- Implemented authentication and access control to ensure secure data handling.
- Automated reporting and monitoring to reduce manual errors and improve transparency in rural schools.

### Tech Stack

- Node.js, Express.js, MongoDB, ESP32, RFID, AI Facial Recognition, REST APIs
